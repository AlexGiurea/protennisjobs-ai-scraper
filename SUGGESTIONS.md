## Product Suggestions

### Notifications
- Daily email or SMS summary for new listings above a configurable threshold (fit score, distance, or keywords).
- Delivery options: email, SMS (e.g., Twilio), or push notification.
- Opt-in filters: minimum fit score, max distance, preferred states, and keyword matches.

### Multi-Source Scraping
- Add more sources (e.g., USTA listings, club career pages) and normalize to the existing data schema.
- Create a pluggable scraper interface so each source can be added independently.
- Track source metadata for each job (source name, scrape timestamp, URL).
